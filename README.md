# Учебный проект

Для локального запуска проекта:

```
npm run i
npm run start
```

Запуск тестов:

```
npm run test
```

Приложение также развернуто на [vercel](https://innotech-project.vercel.app)

Storybook проекта:

```
npm run storybook
```

## Стек:
- React
- TypeScript
- Redux Toolkit, RTK Query
- Jest
- CSS-modules
- Применяется FSD
- Подключен Storybook
